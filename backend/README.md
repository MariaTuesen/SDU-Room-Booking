# SDU Room Booking Backend
This is the backend server for the SDU Room Booking application.

## Prerequisites
- node
- npm

## Install
make sure you are in the `backend` directory, then run:
```bash
npm install
```

## Run
make sure you are in the `backend` directory, then run:
```bash
node server.js
```

The backend server will start on `http://localhost:3000`.
